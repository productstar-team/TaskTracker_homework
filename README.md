# Задача 3. Реализация простой системы управления задачами

Стартап «TaskMaster» разрабатывает новый инструмент для управления проектами и задачами. Вас пригласили в качестве Java-разработчика для создания базовой версии этой системы.

Вы присоединились к команде разработчиков «TaskMaster» на ранней стадии проекта. Ваша задача - разработать основные классы и интерфейсы для системы управления задачами. Эта базовая структура будет использоваться для дальнейшей разработки более сложных функций и интеграции с пользовательским интерфейсом.

В этой задаче вы создадите базовую систему управления задачами, используя изученные концепции.
Создайте класс Task с приватными полями title, description, dueDate и status. Реализуйте геттеры и сеттеры для этих полей.
Создайте перечисление TaskStatus с состояниями задачи (например, TODO, IN_PROGRESS, DONE).
Реализуйте интерфейс Manageable с методами assign(), start(), complete().
Создайте класс Project, который будет содержать коллекцию задач и реализовывать интерфейс Manageable.
Используйте вложенный класс TaskComparator внутри Project для сортировки задач по различным критериям.
Создайте статический класс ProjectStats для хранения статистики по проектам и задачам.
Реализуйте класс TaskManager, который будет управлять несколькими проектами.
Используйте ключевое слово final для методов в классе Task, которые не должны быть переопределены.
Результат работы должен быть представлен в виде Java-файлов с реализацией всех описанных классов и интерфейсов.
